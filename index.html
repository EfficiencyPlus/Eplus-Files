Project Descripton: For KROGER account, every month AR Rep receives requests from Finance to upload billing of at least 4 different projects. Each project has a ZIP file that contains aprox 20 different "detail file S/S". Each "detail file S/S" refers to one system invoice that has aprox 4 tabs with several lines.
AR creates and works in an excel file with different pivot formulas to facilitate and save time in the process of copy and paste to the "template" file. (This was crated because the information provided from the finance team is not aligned to the customer¬¥s requirement.)
AR creates and works on a 3rd excel file named "template" where specific info from the "pivot file" needs to be copied and paste manually in a specific format (values).
After repeating this process for each of the "detail file S/S" at the end, we might end up copying and pasting information making an average between 300 to 1,500 lines.
Before finalizing AR Rep needs to compare and confirm that the total amount of each invoice matches exactly with the amount on the columns (NET amount + TAX amount) included on the template.
If the amount does not match (even by a penny) due to rounding, AR Rep needs to manually change the amounts.
On the ‚Äútemplate‚Äù file, there are 2 columns that needs to be manually fixed in order to show 3 digits. (Store # and division #). In order to have the corresponding division # related to each invoice, AR Rep needs to refer to another excel spreadsheet to validate the information.
Once completed, we save the file ‚Äútemplate‚Äù in 2 formats: .XLS (Excel) and later save as: .CSV file). CSV file would be the final format accepted to be uploaded into the AP portal (I.Payables).
What is the proposed solution? Yes. Since this is billing¬¥s finance responsibility, there should be a person designated to elaborate the final excel and .CSV file.
By doing this, AR Rep can proceed immediately to upload the .CSV file in order to submit invoices, and be able to attach PDF copies along with excel detail S/S into the portal.
